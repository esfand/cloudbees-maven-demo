WHAT IS CLOUDBEES MAVEN DEMO

It is a Maven porting of the default HelloWorld example which is shipped with CloudBees free PaaS offer.
It offers a collection of tips & tricks to achieve a simple J2EE demo hosted by CloubBees just using Apache Maven;
no Cloudbees SDK is needed, no Ant scripts.

For more info, visit http://sourcesense.github.com/cloudbees-maven-demo

I hope this effort will be useful for people who want to start playing with Cloudbees using Maven.
Please also check the official Cloudbees Maven guide - https://cloudbees.zendesk.com/entries/421064-maven-guide
